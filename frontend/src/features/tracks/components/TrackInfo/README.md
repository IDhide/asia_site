# TrackInfo Component

Компонент для отображения информации о текущем треке с кнопками социальных сервисов.

## Структура

- **Название трека** - крупный заголовок
- **Исполнитель** - подзаголовок
- **Кнопка "Тт"** - открывает модальное окно с текстом песни (отображается только если есть текст)
- **6 кнопок соцсетей** - ссылки на музыкальные сервисы:
  - YouTube Music
  - Spotify
  - Apple Music
  - Сбер Звук
  - Яндекс Музыка
  - VK Музыка

## Использование

```tsx
<TrackInfo
  track={currentTrack}
  onLyricsClick={handleOpenModal}
/>
```

## Адаптивность

- **Desktop**: 6 кнопок в ряд
- **Mobile**: 3 кнопки в 2 ряда
