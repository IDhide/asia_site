# TrackDetailModal Component

Модальное окно для отображения деталей трека с текстом песни.

## Структура

### Desktop
- **Левая колонка**: Обложка трека + название + исполнитель
- **Правая колонка**: Заголовок "Текст песни" + текст

### Mobile
- **Сверху**: Обложка трека + название + исполнитель
- **Снизу**: Заголовок "Текст песни" + текст

## Особенности

- Блокирует скролл body при открытии
- Закрывается по клику на overlay
- Закрывается по нажатию Escape
- Sticky позиционирование обложки на desktop
- Анимация появления

## Использование

```tsx
<TrackDetailModal
  track={selectedTrack}
  onClose={handleClose}
/>
```
