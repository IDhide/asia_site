## Состояния слайдера

### Неактивные слайды
```
┌────────┐
│        │
│ blur   │  ← opacity: 0.4
│ 0.4    │  ← filter: blur(4px)
│        │  ← scale: 0.85
└────────┘
```

### Активный слайд
```
┌──────────────┐
│              │
│   ОБЛОЖКА    │  ← opacity: 1
│   ТРЕКА      │  ← filter: blur(0)
│   (клик!)    │  ← scale: 1
│              │  ← cursor: pointer
└──────────────┘
```

### Hover на активном слайде
```
┌──────────────┐
│              │
│   ОБЛОЖКА    │  ← scale: 1.02
│   ТРЕКА      │  ← transition: smooth
│   (клик!)    │
│              │
└──────────────┘
```

## Анимации

### Появление модалки
```
0%   → opacity: 0, translateY(40px)
100% → opacity: 1, translateY(0)
Duration: 0.4s
Easing: cubic-bezier(0.22, 0.61, 0.36, 1)
```

### Hover на кнопках
```
Normal → Hover
scale: 1 → 1.05
translateY: 0 → -2px
Duration: 0.22s
```

### Переключение слайдов
```
Duration: 520ms
Effect: coverflow 3D
Rotate: 50deg
Depth: 120px
```
