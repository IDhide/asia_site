# Shop Feature

Компоненты для магазина с минималистичным дизайном.

## Компоненты

### ProductPage
Страница детального просмотра товара.

**Структура:**
- Десктоп: 50/50 (фото слева, информация справа)
- Мобайл: фото сверху, информация снизу
- Под карточкой: сетка других товаров

**Использование:**
```tsx
<ProductPage
  product={product}
  allProducts={allProducts}
  onClose={() => router.back()}
/>
```

### ProductsGrid
Универсальная сетка товаров (только картинки).

**Использование:**
```tsx
// Все товары
<ProductsGrid products={products} />

// Все кроме текущего
<ProductsGrid products={products} excludeId={currentId} />
```

## Страницы

- `/shop` - сетка всех товаров
- `/shop/[slug]` - детальная страница товара

## Особенности

- Темная тема (#000)
- Сетка 3x на десктопе, 2x на мобайле
- Без отступов между элементами
- Интеграция с DataContext и корзиной
