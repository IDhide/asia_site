## Как использовать

### 1. Навигация по трекам
- **Свайп** влево/вправо на слайдере
- **Клавиши** ← → на клавиатуре
- **Клик** на боковые обложки

### 2. Открыть детали трека
- **Клик** на центральную обложку
- **Клик** на кнопку "Тт" (если есть текст)

### 3. Модальное окно
- **Desktop**: обложка слева, текст справа
- **Mobile**: обложка сверху, текст снизу
- **Закрыть**: Escape, клик вне окна, кнопка ✕
