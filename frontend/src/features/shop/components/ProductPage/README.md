# ProductPage Component

Компонент страницы товара с дизайном согласно референсу.

## Структура

### Десктоп
- **Левая часть**: Квадратное фото с крестиком слева вверху и кнопками навигации по центру снизу
- **Правая часть**: Название, цена, кнопка "КУПИТЬ", размеры, галерея миниатюр, доп. информация

### Мобайл
- Фото сверху
- Информация снизу

### Под карточкой товара
- Сетка с другими товарами (все кроме текущего)

## Использование

```tsx
import { ProductPage } from '@/features/shop';

<ProductPage
  product={product}
  allProducts={allProducts}
  onClose={() => router.back()}
/>
```

## ProductsGrid Component

Универсальный компонент сетки товаров (3 колонки на десктопе, 2 на мобайле).

```tsx
import { ProductsGrid } from '@/features/shop';

// На странице /shop (все товары)
<ProductsGrid products={products} />

// Под товаром (все кроме текущего)
<ProductsGrid products={products} excludeId={currentProductId} />
```

## Особенности

- Темная тема (#000, #0a0a0a)
- Адаптивный дизайн
- Горизонтальный скролл миниатюр с градиентами
- Интеграция с корзиной
- Сетка без отступов между элементами
